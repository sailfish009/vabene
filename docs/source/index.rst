============
Introduction
============

GitHub: https://www.github.com/lukasturcani/vabene

:mod:`vabene` is a Python library for creating molecular graphs,
which obey user-defined valence restrictions.

Installation
------------

To get :mod:`vabene`, you can install it with pip::

    $ pip install vabene


Examples
--------

There are bunch of examples you can see by clicking on the
``Next`` button, or looking at the sidebar.
